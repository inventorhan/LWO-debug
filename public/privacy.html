LWO 물류 분석 개인정보처리방침

시행일: 2026년 4월 30일 · 최종 수정일: 2026년 4월 30일

본 개인정보처리방침은 LWO 물류 분석(이하 "본 앱")이 사용자의 개인정보를 어떻게 처리하는지를 안내하기 위해 작성되었습니다.

1. 수집·처리하는 정보

본 앱은 사용자의 개인정보를 외부 서버로 전송하거나 수집하지 않습니다. 본 앱이 사용하는 모든 데이터는 사용자의 기기 내부에서만 저장·처리됩니다.

구분처리 정보저장 위치처리 목적
측정 데이터 작업자 이름, 측정 시각, 운반 시간, 면적, AMR 산식 입력값 등 기기 내 localStorage 물류 작업 분석 결과 산출 및 표시
사진 사용자가 직접 촬영하거나 선택한 운반/적재 사진 기기 내 localStorage (Base64) 측정 보조 자료 첨부
JSON / Excel 파일 사용자가 명시적으로 저장한 분석 결과 파일 사용자 기기 다운로드 폴더 분석 결과 내보내기 / 공유
중요: 위 정보는 사용자의 기기를 떠나지 않습니다. 개발자, Anthropic, Google 또는 어떤 제3자도 본 앱을 통해 사용자의 데이터에 접근할 수 없습니다.

2. 사용 권한 및 사용 사유

권한사용 목적
android.permission.CAMERA 운반 작업·E/V 적재·구역 사진을 즉시 촬영하여 측정 카드에 첨부하기 위함
android.permission.READ_MEDIA_IMAGES
android.permission.READ_EXTERNAL_STORAGE		 사용자가 갤러리에서 기존 사진을 선택하여 첨부하기 위함
android.permission.WRITE_EXTERNAL_STORAGE
(SDK 29 이하)		 분석 결과 JSON·Excel 파일을 내보내기 위함
android.permission.INTERNET WebView 기반 앱 구동을 위한 시스템 리소스 접근 (외부 통신은 수행하지 않음)

3. 정보 보유 및 삭제

본 앱이 저장한 데이터는 다음 방법으로 즉시 삭제할 수 있습니다.

  • 앱 헤더의 🔄 초기화 버튼을 눌러 모든 입력 데이터를 즉시 삭제
  • 안드로이드 설정 → 앱 → LWO 물류 분석 → 저장공간 → 데이터 삭제
  • 앱을 제거하면 기기 내 모든 관련 데이터가 함께 삭제됩니다

4. 제3자 제공 및 광고

본 앱은 어떠한 제3자에게도 사용자의 정보를 제공하지 않으며, 광고 SDK를 포함하지 않습니다.

5. 아동 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

6. 보안

본 앱이 저장하는 모든 데이터는 사용자 기기의 운영체제가 제공하는 샌드박스 내에 보관됩니다. 사용자가 안드로이드 화면 잠금·생체 인증을 활성화하면 추가적인 보안 계층이 적용됩니다.

7. 정책 변경

본 방침이 변경될 경우 본 페이지에서 시행일과 함께 안내합니다. 중요한 변경이 있을 경우 앱 업데이트 시 출시 노트로 안내합니다.

## v1.2.7 (2026-05-21)

- ✨ **신규 모듈 3종 추가**
  - `물류 적정 인원`: 피킹·이동·로딩/언로딩 시간과 운반 횟수 기반 필요 인원 산출
  - `물류 창고 면적`: CMDT별 물동·용기 면적·DIO·여유율 기반 창고 면적(m²/평) 산출
  - `물류 자동화율`: 자동화 적용 Item 및 Re-Handling Item 기준 자동화율/Re-Handling율 산출
- 📊 **작업자 부하율 상세 시간 그래프 추가** — 회차별/작업자별, 라인형/박스형 보기 지원
- 📊 **작업자 전체 인원 대시보드 보강** — 각 그래프 왼쪽에 평균 기준 막대를 추가해 개인별 결과와 평균을 바로 비교
- 🧮 **재고 보관량 산식 보정** — 부족/잉여 차이값을 절대값으로 적용해 최종 적정 재고 및 Space 산출
- 🧮 **신규 산출 모듈 산식/레이아웃 보정** — 물류 적정 인원은 여유율(%) 기준으로 정리하고, 창고 면적은 순차 입력 흐름을 유지하면서 2열 입력/전체 폭 결과 카드로 개선
- 🎨 **모바일 탭 표시 개선** — 신규 항목명을 `물류인원`, `창고면적`, `자동화율`로 축약 표시하고 자동화율 아이콘을 조정
- 🖼 **면적 효율 사진 영역 개선** — PC 화면에서 공장 사진이 입력 영역과 겹치지 않도록 레이아웃 보정
- 🖥 **Windows 설치 안정화** — 업데이트 설치 전 백그라운드 `LWO.exe` 자동 종료 및 구버전 uninstaller 실패 우회
- 📑 **Excel 내보내기 확장** — 신규 3개 모듈을 포함해 총 9개 시트로 내보내기
- 📖 **문서 전체 업데이트** — 앱 내 도움말, 교육용 매뉴얼, README, 프로젝트 구조 문서를 9개 모듈 기준으로 동기화

## 버전 번호 규칙 (Semantic Versioning)

`vMAJOR.MINOR.PATCH`

- **MAJOR** — 큰 구조 변경, 데이터 호환성 깨짐 (예: 2.0.0)
- **MINOR** — 신기능 추가 (예: 1.2.0)
- **PATCH** — 버그 수정·작은 개선 (예: 1.1.13)

각 배포마다 `versionCode`는 +1, `versionName`은 위 규칙으로 변경.
